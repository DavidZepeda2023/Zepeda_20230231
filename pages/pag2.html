RJ45

Descripción: El conector RJ45 es un estándar de conectores modulares que se utiliza principalmente en redes Ethernet. Tiene ocho pines, que corresponden a los ocho conductores del cableado UTP (Par Trenzado No Blindado).

Uso: Se utiliza para conectar dispositivos de red, como computadoras, switches, y routers, a través de cables de red como el cable UTP.

Jacks o Keystone

Descripción: Los jacks Keystone son conectores modulares que se instalan en placas de pared, patch panels o cajas de conexión. Están diseñados para alojar conectores modulares, como los de RJ45.

Uso: Facilitan la conexión y desconexión de cables de red en instalaciones fijas.

Cable UTP

Descripción: El cable UTP (Unshielded Twisted Pair) es un tipo de cable de red que consiste en pares de hilos de cobre trenzados sin apantallamiento. Es comúnmente utilizado en redes Ethernet.

Uso: Transmisión de datos en redes de área local (LAN).

Patch Panel

Descripción: Un patch panel es un panel de interconexión donde se terminan y gestionan los cables de red. Los cables UTP se conectan en la parte trasera del panel, mientras que la parte frontal tiene puertos RJ45 para la conexión de dispositivos o patch cords.

Uso: Organizar y gestionar múltiples cables de red en una instalación, permitiendo cambios y conexiones rápidas.

Patch Cord

Descripción: Un patch cord es un cable de red flexible con conectores RJ45 en ambos extremos. Se utiliza para conectar dispositivos a un patch panel o directamente a equipos de red.

Uso: Realizar conexiones temporales o cambiar la configuración de la red rápidamente.

Fibra Óptica

Descripción: La fibra óptica es un medio de transmisión que utiliza filamentos de vidrio o plástico para transmitir datos como pulsos de luz. Ofrece alta velocidad y gran ancho de banda en largas distancias.

Uso: Transmisión de datos en redes de alta velocidad, como las redes troncales de Internet, redes de larga distancia y algunas LAN.

Canaletas

Descripción: Las canaletas son conductos rígidos o flexibles que se utilizan para encaminar y proteger cables eléctricos y de datos en instalaciones fijas.

Uso: Organización, protección y ocultación de cables en instalaciones eléctricas y de redes.



Conceptos 2

Switch

Descripción: Un switch es un dispositivo de red que conecta múltiples dispositivos dentro de una misma red local (LAN). Funciona en la capa 2 del modelo OSI, utilizando direcciones MAC para reenviar datos al dispositivo de destino correcto dentro de la red.

Uso: Es utilizado para crear redes LAN al conectar dispositivos como computadoras, impresoras y servidores, permitiendo la comunicación y el intercambio de datos entre ellos.

Router

Descripción: Un router es un dispositivo de red que se encarga de dirigir y enrutar el tráfico de datos entre diferentes redes, como entre una red local (LAN) e Internet. Opera en la capa 3 del modelo OSI, utilizando direcciones IP para determinar la mejor ruta para enviar los datos.

Uso: Se utiliza para conectar redes distintas, como una red doméstica a Internet, y gestionar el tráfico de datos entre ellas. También proporciona funciones de seguridad como firewall y NAT (Network Address Translation).

Tarjeta de red (NIC - Network Interface Card)

Descripción: La tarjeta de red es un componente de hardware que permite a un dispositivo, como una computadora o un servidor, conectarse a una red. Puede ser tanto inalámbrica (Wi-Fi) como cableada (Ethernet) y se conecta a la placa base del dispositivo.

Uso: Facilita la comunicación en una red, permitiendo la transferencia de datos entre dispositivos dentro de la red y hacia otras redes, como Internet.

Interruptor inteligente WiFi

Descripción: Un interruptor inteligente WiFi es un dispositivo que permite controlar la alimentación eléctrica de un dispositivo o luz a través de una conexión WiFi. Puede ser controlado remotamente a través de aplicaciones móviles o comandos de voz.

Uso: Se utiliza en hogares inteligentes para automatizar y controlar dispositivos eléctricos, como luces o electrodomésticos, de manera remota y programable.

Access Point (Punto de Acceso)

Descripción: Un Access Point es un dispositivo de red que permite que dispositivos inalámbricos (como teléfonos, tabletas y computadoras portátiles) se conecten a una red cableada. Extiende la cobertura de una red WiFi, conectando los dispositivos inalámbricos a la red LAN.

Uso: Se utiliza para aumentar la cobertura de una red WiFi, permitiendo que más dispositivos se conecten a la red sin necesidad de cables físicos. Es común en entornos empresariales o en hogares grandes donde la señal WiFi del router no es suficiente.

Conceptos 3

Ponchadora

Descripción: La ponchadora es una herramienta manual utilizada para conectar o "ponchar" cables en conectores, como los de tipo IDC (Insulation Displacement Connector), típicamente en punch-down blocks o patch panels. Funciona al insertar el cable en el conector y luego cortar el exceso, asegurando una conexión sólida y permanente.

Uso: Se utiliza principalmente en instalaciones de redes de datos y telecomunicaciones para conectar cables de red (como el cable UTP) en bloques de conexión y paneles de parcheo.

Crimpadora

Descripción: La crimpadora es una herramienta diseñada para unir un conector (como un conector RJ45) a un cable (como un cable de red UTP). La crimpadora comprime el conector alrededor del cable, asegurando tanto una conexión física como eléctrica.

Uso: Es utilizada para preparar cables de red, especialmente en la creación de cables Ethernet personalizados, crimpando conectores RJ45 o RJ11 en los extremos de los cables.

Desforradora

Descripción: La desforradora es una herramienta que se usa para retirar el recubrimiento exterior de un cable, exponiendo los conductores internos sin dañarlos. Es ajustable para adaptarse a diferentes diámetros de cables.

Uso: Se emplea en la preparación de cables para la instalación de conectores, permitiendo pelar el aislamiento de los cables con precisión antes de realizar la conexión o crimpado.

Tester/Probador de Cable

Descripción: Un tester o probador de cables es un dispositivo que verifica la continuidad, la integridad y la correcta terminación de cables, como cables Ethernet o de teléfono. Puede detectar errores como cables cruzados, roturas, conexiones incorrectas y cortocircuitos.

Uso: Se utiliza para comprobar que los cables de red estén conectados correctamente y funcionen de manera adecuada, asegurando que los datos se transmitan sin problemas en una red. Es esencial en el diagnóstico de problemas de conectividad y en la validación de nuevas instalaciones.
